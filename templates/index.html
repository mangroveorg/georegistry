Introduction:

Welcome to the Georegistry, the geospatial health index service developed by the Earth Institute. What makes georegistry special is that it is deisgned to build high-quality, vetted health information from multiple sources. Georegistry provides a standardized way to collect and store health data that is often kept in spreadsheets in incompatiable formats. To see a client example visit http://www.georegistry.org. You can also have a look at the developer documentation, or request access to the application programming interface(API). Currently, we are only accepting input from governments and NGOs sources for the following countries:

  • Nigeria
  • Ghana

Georegistry Capabilities:

  • Store and search multiple spellings or langauages for a single item
  • Duplicate detection
  • Flag items as "offical" or government verified
  • Historical storage and search
  • Spreadsheet import tools data
  • Developer Friendly: RESTful API serving GeoJSON, open source, Client Examples with Source Code
  • Commect via mobile decvices, web, xforms, or other application
